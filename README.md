# ЖКС Загребенки — сайт Жилищно-коммунальной службы

Статический информационный сайт для ЖКС города Загребенки. Адаптивный, современный, минималистичный дизайн. Вся динамическая информация хранится в файле `data.json` — редактировать данные можно без знания программирования.

## Структура файлов

```
index.html      — главная страница (лендинг с поиском, превью разделов)
news.html       — все новости и объявления
districts.html  — все районные участки со списками домов
outages.html    — полный график плановых отключений
contacts.html   — контакты, реквизиты, схема проезда
privacy.html    — политика конфиденциальности
style.css       — все стили (CSS-переменные, сетки, анимации, адаптивность)
common.js       — общие функции + встроенные данные (запасной вариант)
script.js       — логика только главной страницы
data.json       — данные организации (дублируются в common.js)
README.md       — этот файл
```

## Как запустить

Сайт полностью статический. Просто откройте `index.html` в браузере — данные уже встроены в `common.js`, никакой сервер не нужен.

## Как редактировать данные

Откройте файл `data.json` (или константу `EMBEDDED_DATA` в `common.js`) в любом текстовом редакторе (Блокнот, VS Code, Notepad++ и т.д.). Если сайт открывается через `file://`, данные берутся из `common.js`, поэтому меняйте данные в обоих файлах.

### Основные поля

- `organization` — полное название организации
- `emergencyPhone` — номер аварийной диспетчерской (отображается в шапке)
- `contacts` — контакты и реквизиты

### Районы (districts)

Массив объектов. Каждый район содержит:
- `name` — название участка
- `address` — адрес участка
- `phone` — телефон диспетчера
- `head` — ФИО начальника
- `headSchedule` — график приёма
- `streets` — список улиц (для отображения)
- `homes` — полный список домов (используется для поиска по адресу)

### Новости (news)

Массив объектов. Каждая новость содержит:
- `date` — дата в формате YYYY-MM-DD
- `title` — заголовок
- `short` — краткое описание (на карточке)
- `full` — полный текст (в модальном окне)
- `urgent` — true/false (срочное объявление)

### Отключения (outages)

Массив объектов. Каждая запись содержит:
- `district` — район
- `date` — дата
- `time` — время
- `type` — тип (вода, электричество, газ)
- `addresses` — адреса

## Как изменить email для получения заявок

Откройте `script.js` и найдите в начале файла:

```js
const CONFIG = {
  dataUrl: 'data.json',
  formEmail: 'info@zhabg.ru',       // ← замените на ваш email
  formEndpoint: 'https://formspree.io/f/yourFormID', // ← замените на ID Formspree
};
```

По умолчанию формы имитируют отправку (показывают сообщение об успехе). Чтобы реально отправлять данные на email, используйте сервис [Formspree](https://formspree.io) — зарегистрируйтесь, создайте форму и вставьте полученный ID в `formEndpoint`.

## Кастомизация стилей

Все цвета и основные размеры заданы через CSS-переменные в начале файла `style.css`:

```css
--color-primary: #1A3B5C;       /* основной тёмно-синий */
--color-accent: #2B7BEE;        /* акцентный синий */
--color-bg-secondary: #F7F9FC;  /* фоновый серый */
--radius: 4px;                  /* скругление углов */
```

## Поддержка

По вопросам работы сайта обращайтесь в ЖКС Загребенки по телефону или email, указанным на сайте.
